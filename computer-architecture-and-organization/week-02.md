**Question 6. What is the correct MIPS32 representation of the following high-level code?**

    if (x < y)
      Result = x + y;
    else
      Result = x - y;

a)

    slt $t0, $s0, $s1  
    beq $t0, $zero, Else  
    add $s2, $s0, $s1  
    j Exit  
    Else: sub $s2, $s0, $s1  
    Exit: ...

b) Similar to (a) but with different branch\
c) bge $s0, $s1, Else
